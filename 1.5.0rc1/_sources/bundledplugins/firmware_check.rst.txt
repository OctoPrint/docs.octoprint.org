Firmware Check
==============

.. versionadded:: 1.3.7

The firmware check plugin - while considered bundled with OctoPrint - is a separate project on its own release
cycle and documented on `its Github repository <https://github.com/OctoPrint/OctoPrint-FirmwareCheck>`_.

It was formerly called "Printer Safety Check" and directly bundled with OctoPrint.
